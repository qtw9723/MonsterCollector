# GDD v0.1

## 기본 정보
- **장르**: 탑뷰 핵앤슬래시 RPG + 로그라이크 + 가챠 컬렉션
- **엔진**: Unity (C#)
- **플랫폼**: 미정 (솔로 플레이 전용)

---

## 핵심 루프
```
파밍 → 아이템으로 티켓 제작 → 가챠 → 몬스터 획득 → 장착 → 더 깊은 던전
```

---

## 몬스터 시스템
- 플레이어 주변에서 자동으로 행동 (공격 / 버프 / 탱킹 / 스탯 보조)
- 기본 장착 슬롯 3개 → 캐릭터/본거지 성장으로 확장 가능
- 등급: **S / A / B / C / D**

---

## 플레이어 성장
- 레벨업 + 아이템 장착
- 스킬트리 없음
- 던전: 로그라이크 무한 던전 구조

---

## 아이템 시스템
- 디아블로2 참고
- 등급: Normal / Magic / Rare / Unique / Set
- 파밍한 아이템 → 조합 → 가챠 티켓 제작

---

## 가챠 시스템
- 메인: 티켓 방식 (1뽑 / 10뽑), 천장 포함 → [상세](GACHA_REDESIGN_PLAN.md)
- 서브: 숫자 맞추기 미니게임 (기존 코드 유지)

---

## 본거지 시스템
- 몬스터 슬롯 확장
- 제작소 (아이템 조합)
- 훈련장
- 창고
