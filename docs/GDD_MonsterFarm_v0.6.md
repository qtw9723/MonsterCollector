## 1. 게임 개요

### 1.1 한줄 소개
디아블로2 스타일의 탑뷰 핵앤슬래시 파밍 루프에 가챠 몬스터 컬렉션 시스템을 결합한 로그라이크 RPG. 마왕에게 궤멸당한 지상을 되찾기 위해 생존자와 몬스터를 규합하는 이야기.

### 1.2 핵심 컨셉
- 파밍한 아이템으로 티켓을 제작 → 가챠로 몬스터 획득
- 획득한 몬스터를 최대 3마리 장착 → 공격/버프/탱킹/스탯 보조
- 몬스터 조합으로 빌드 다양성 극대화
- 로그라이크 무한 던전 — 파밍 루프의 끝이 없는 구조
- 1회차에서 히든엔딩 조건 달성 시 일반 엔딩 이후 추가 루트 오픈

### 1.3 레퍼런스 게임

| 게임 | 참고 요소 |
|------|---------|
| 디아블로 2 | 탑뷰 핵앤슬래시, 아이템 파밍 루프, 빌드 다양성 |
| 포켓몬 | 몬스터 컬렉션 + 장착 + 역할 분담 |
| 패스 오브 엑자일 | 깊은 아이템 시스템, 무한 파밍 동기 |
| 슬레이 더 스파이어 | 로그라이크 던전 진행 구조 |

## 2. 스토리 & 세계관

### 2.1 세계관 설정
- **장르**: 밝은 클리셰 판타지 세계관
- **지상 세계**: 인간, 엘프, 드워프 등 여러 종족이 공존하던 평화로운 세계
- **지하 세계**: 마왕이 다스리는 어둠의 영역. 다양한 지하 몬스터들이 서식
- **몬스터**: 지상/지하 모두에 원래부터 존재하는 자연 생물. 적이 될 수도 동료가 될 수도 있음

### 2.2 핵심 설정 — 봉인된 재앙
지상 깊은 곳에는 고대의 재앙이 봉인되어 있다. 이 재앙은 지상과 지하 양쪽을 모두 멸망시킬 수 있는 존재. 마왕은 수천 년간 이 봉인을 지켜왔으나, 지상인들이 무지하게 봉인에 접근하기 시작하자 이를 막기 위해 지상을 침략했다. 즉, 마왕의 침략은 악의가 아닌 절박함에서 비롯된 것.

### 2.3 메인 스토리 흐름

| 단계 | 내용 |
|------|------|
| 프롤로그 | 마왕의 군대가 지상을 침략. 왕국은 궤멸되고 주인공은 생존자 중 한 명으로 깨어남 |
| Act 1 — 규합 | 폐허가 된 지상을 탐색하며 생존자 NPC를 모으고, 우연히 첫 번째 몬스터를 부하로 만듦 |
| Act 2 — 역습 | 생존자 거점(본거지)을 구축하고 몬스터들을 규합. 지하 세계 입구를 향해 진격 |
| Act 3 — 대결 | 마왕과 첫 대결. 마왕을 쓰러뜨리지만 지하 세계로 도망침 |
| 일반 엔딩 | 마왕을 몰아내고 지상 재건 시작. 그러나 진짜 위협은 아직 봉인 속에... |
| 히든 엔딩 루트 | 조건 달성 시 일반 엔딩 직후 추가 루트 오픈 → 지하 세계 입장 → 마왕 추격 → 진실을 알고 협력 → 봉인된 재앙(히든 보스) 처치 → 진정한 평화 달성 + 전용 파밍처 오픈 |

### 2.4 엔딩 분기 구조

- 힌트는 곳곳에 숨겨져 있음 — 아이템 설명, NPC 대화, 환경 묘사 등으로 암시
- 플레이어가 스스로 눈치채고 조건을 달성해야 함 (친절한 안내 없음)

```
조건 달성 O  →  마왕 격퇴  →  일반 엔딩  →  추가 루트 오픈  →  히든 엔딩
조건 달성 X  →  마왕 격퇴  →  일반 엔딩으로 마무리
```

**히든 엔딩 조건 (예시)**

| 조건 유형 | 예시 | 힌트 방식 |
|---------|------|---------|
| 특정 사이드퀘스트 완료 | 마왕의 과거를 기록한 고대 현자 NPC 퀘스트 완료 | 현자 NPC 대화에서 단편적으로 암시 |
| 특정 아이템 소지 | 고대 봉인석 수집 (지상 던전 곳곳에 숨겨짐) | 아이템 설명 텍스트에 단서 |
| 특정 몬스터 소집 | 지상 전령 몬스터 영입 (1회차에서 달성 가능) | 특정 NPC 대화에서 전령의 존재 암시 |

> ※ 3가지 조건 모두 1회차에서 달성 가능. 힌트는 곳곳에 숨겨져 있으며 플레이어가 직접 찾아야 함.

**일반 엔딩**
- 마왕을 쓰러뜨리고 지상으로 귀환
- 지상 재건을 시작하지만 어딘가 찜찜한 결말
- 엔딩 크레딧 후 '무언가 봉인이 흔들리고 있다...' 텍스트로 마무리

**히든 엔딩 (진엔딩)**
- 일반 엔딩 직후 지하 세계 입장 루트 오픈
- 마왕 추격 → 진실(봉인된 재앙의 정체)을 알게 됨
- 마왕과 협력하여 봉인된 재앙(히든 보스) 처치
- 히든 보스 전용 파밍처 오픈 — 일반 엔딩에서는 접근 불가한 드롭 아이템
- 지상과 지하의 진정한 화해, 완전한 평화 달성

### 2.5 뉴게임+ (히든 엔딩 클리어 이후)
- 히든 엔딩 클리어 시 2회차(뉴게임+) 해금
- 2회차부터 지하 세계 몬스터 영입 가능 — 기존 지상 몬스터보다 강한 게 아닌 완전히 다른 스타일의 몬스터
- 지하 몬스터는 지상 몬스터와 수치는 비슷하지만 고유한 메커닉/역할로 빌드 다양성 확장
- 지상 몬스터 + 지하 몬스터 조합으로 기존에 없던 새로운 빌드 가능
- 지하 세계 전용 던전/퀘스트 오픈

### 2.6 주요 등장인물

| 캐릭터 | 역할 | 설명 |
|------|------|------|
| 주인공 | 플레이어 캐릭터 | 생존자. 장비 자유. 우연히 첫 몬스터를 부하로 만들며 모험 시작 |
| 마왕 | 메인 빌런 (반전) | 지상을 침략했지만 사실 재앙의 봉인을 지키기 위한 절박한 선택. 히든엔딩에서 협력자로 전환 |
| 생존자 NPC | 동료 | 각자 사연을 가진 생존자들. 사이드퀘스트와 연결. 본거지에 거주 |
| 고대 현자 (NPC) | 핵심 정보 제공자 | 봉인의 진실을 아는 유일한 존재. 히든 엔딩 퀘스트의 핵심. 대화에서 단편적 힌트 제공 |

## 3. 핵심 게임플레이 루프

### 3.1 메인 루프
```
던전 입장
  → 몬스터 처치 & 아이템 드롭
  → 아이템 조합으로 티켓 제작
  → 가챠 돌리기
  → 몬스터 획득/강화
  → 더 깊은 던전 입장
```

### 3.2 서브 루프
- **본거지(홈베이스) 성장** — 몬스터 슬롯 확장, 편의 시설 업그레이드
- **캐릭터 레벨업** — 레벨에 따른 스탯 성장
- **아이템 장착** — 장비 세팅으로 캐릭터 기본 스펙 강화
- **사이드퀘스트** — NPC 동료 관련 퀘스트, 히든엔딩 조건과 연결

## 4. 플레이어 캐릭터

### 4.1 성장 방식

| 시스템 | 내용 |
|------|------|
| 레벨업 | 경험치 획득 → 레벨업 → 기본 스탯(HP, 공격력, 방어력 등) 증가 |
| 아이템 장착 | 무기/방어구/악세서리 장착으로 스탯 강화 (스킬트리 없음) |
| 몬스터 장착 | 최대 3슬롯(확장 가능)에 몬스터를 장착해 추가 능력 획득 |

### 4.2 몬스터 슬롯
- **기본 슬롯**: 3슬롯
- **확장 조건**: 캐릭터 레벨 달성 OR 본거지 업그레이드
- **최대 슬롯 수**: 추후 결정 (6슬롯 이상 검토)

## 5. 몬스터 시스템

### 5.1 등급 체계

| 등급 | 희귀도 | 가챠 확률 (예시) | 특징 |
|------|------|-------------|------|
| S | 전설 | 1% | 고유 패시브 보유, 최상위 스탯 |
| A | 희귀 | 5% | 강력한 스킬, 높은 스탯 |
| B | 일반희귀 | 15% | 준수한 성능, 범용성 높음 |
| C | 일반 | 35% | 기초 능력, 합성 재료로 활용 |
| D | 하급 | 44% | 낮은 스탯, 티켓 재료 |

### 5.2 역할 유형

| 역할 | 설명 | 예시 능력 |
|------|------|---------|
| 딜러 (Attacker) | 플레이어 주변에서 적을 직접 공격 | 주기적 범위 공격, 단일 타겟 집중 딜 |
| 버퍼 (Buffer) | 플레이어 스탯/스킬을 강화 | 공격력 +%, 공격속도 증가, 크리티컬 버프 |
| 디버퍼 (Debuffer) | 적의 스탯/행동을 약화 | 이동속도 감소, 방어력 감소, 상태이상 |
| 탱커 (Tank) | 플레이어 대신 피해 흡수 | 도발, 보호막 생성, 피해 분산 |
| 스탯 강화 (Passive) | 플레이어 기본 스탯을 패시브로 상승 | HP 증가, 마나 회복, 이동속도 증가 |

### 5.3 몬스터 출처별 분류

| 분류 | 영입 가능 시점 | 특징 |
|------|------------|------|
| 지상 몬스터 | 1회차부터 가능 | 지상 던전에서 파밍한 티켓으로 가챠 |
| 지하 몬스터 | 히든 엔딩 클리어 후 2회차(뉴게임+)부터 가능 | 수치는 지상 몬스터와 유사. 고유 메커닉/역할로 빌드 다양성 확장. 지상 몬스터와 조합해 새로운 빌드 가능 |

## 6. 아이템 & 티켓 시스템

### 6.1 아이템 등급
- Normal / Magic / Rare / Unique / Set — 등급별 드롭률과 스탯 차이 존재
- 접두/접미 옵션 조합으로 아이템마다 고유 스탯 생성

### 6.2 티켓 제작

| 티켓 등급 | 필요 재료 (예시) | 획득 가능 몬스터 등급 |
|---------|-------------|---------------|
| S급 티켓 | Unique 아이템 3개 + 특수 재료 | S, A |
| A급 티켓 | Rare 아이템 5개 | A, B |
| B급 티켓 | Magic 아이템 10개 | B, C |
| C급 티켓 | Normal 아이템 20개 | C, D |

> ※ 재료 수치는 밸런싱 테스트 후 조정 예정

## 7. 던전 시스템

### 7.1 로그라이크 무한 던전
- 층수가 깊어질수록 몬스터 강도 및 드롭 아이템 등급 상승
- 매 런마다 던전 레이아웃, 몬스터 배치, 이벤트 랜덤 생성
- 사망 시 던전 초기화 (로그라이크 방식)
- 현재 최고 기록(층수)은 별도 저장

### 7.2 던전 종류

| 던전 | 해금 조건 | 특징 |
|------|---------|------|
| 지상 던전 | 기본 제공 | 지상 몬스터, Normal~Unique 아이템 드롭 |
| 지하 던전 | 히든 엔딩 클리어 후 2회차부터 | 지하 몬스터, 더 높은 등급 아이템, S급 티켓 재료 |

### 7.3 던전 구성 요소

| 요소 | 내용 |
|------|------|
| 일반 방 | 몬스터 처치 → 아이템 드롭 |
| 엘리트 방 | 강화된 몬스터, 더 좋은 드롭 |
| 보스 방 | 각 층 구간별 보스, 특수 드롭 |
| 이벤트 방 | 랜덤 이벤트 (상점, 힐링, 선택지 등) |
| 상점 방 | 골드로 아이템 구매 |

## 8. 본거지 (홈베이스) 시스템

### 8.1 개요
플레이어가 던전 밖에서 관리하는 거점. 생존자 NPC들이 모이는 장소이자 각종 업그레이드의 허브.

### 8.2 주요 시설

| 시설 | 기능 | 업그레이드 효과 |
|------|------|-------------|
| 몬스터 우리 | 보유 몬스터 관리 | 수용 가능 몬스터 수 증가, 슬롯 확장 |
| 가챠 기계 | 티켓으로 가챠 실행 | 보너스 확률 이벤트 |
| 제작소 | 아이템 → 티켓 변환 | 제작 레시피 확장 |
| 훈련장 | 몬스터 경험치 수동 획득 | 훈련 효율 증가 |
| 창고 | 아이템 보관 | 보관 칸수 증가 |
| NPC 거처 | 생존자 NPC 주거 공간 | NPC 퀘스트 해금, 대화 콘텐츠 추가 |

## 9. 개발 우선순위 (로드맵)

### Phase 1 — 코어 프로토타입
- 플레이어 이동 & 탑뷰 카메라
- 기본 전투 (근접/원거리 공격)
- 적 AI (추적, 공격 — A* Pathfinding Project 활용)
- 기본 아이템 드롭 & 인벤토리
- 간단한 던전 생성 (프로시저럴)

### Phase 2 — 몬스터 시스템
- 가챠 UI 및 로직
- 몬스터 슬롯 장착 시스템
- 몬스터 AI (플레이어 추종 + 역할별 행동)
- 등급별 가챠 확률 구현

### Phase 3 — 스토리 & 깊이 추가
- 메인 스토리 퀘스트 구현 (프롤로그 ~ Act 3)
- 생존자 NPC 사이드퀘스트
- 히든엔딩 조건 추적 시스템 (달성 여부 내부 저장)
- 일반 엔딩 / 히든 엔딩 분기 구현
- 티켓 제작 시스템
- 본거지 시스템
- 아이템 옵션 시스템 (접두/접미)

### Phase 4 — 뉴게임+ & 폴리싱
- 히든 엔딩 클리어 후 뉴게임+ 해금 시스템
- 지하 세계 던전 및 지하 몬스터
- 몬스터 빌드 다양성 밸런싱
- UI/UX 개선
- 사운드 & 이펙트
